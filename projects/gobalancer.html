GoBalancer

Layer 7 reverse proxy load balancer built in Go featuring Round Robin scheduling, backend health checks and fault tolerant traffic routing.

GitHub

Overview

GoBalancer is a reverse proxy load balancer built from scratch to understand how production systems distribute traffic across multiple backend servers.

Request Lifecycle

  1. Client sends HTTP request
  2. Load balancer receives request
  3. Round Robin selects backend
  4. Health status verified
  5. Request forwarded
  6. Response returned to client
